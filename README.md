## Jekyll Serve
### Windows
1. open cmd as administrator admission

2. install ruby
```cmd
choco install ruby --version 2.7.7.1
```

3. install MSYS2 and MINGW
```cmd
ridk install 3
```

3. gem package install
```cmd
bundle install
```

4. start serving
```cnd
bundle exec jekyll serve
```
